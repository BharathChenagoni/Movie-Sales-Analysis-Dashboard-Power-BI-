🎬 Movie Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project focuses on analyzing movie industry data to identify key factors influencing box office success. Using Power BI, an interactive dashboard was developed to explore trends in revenue, ratings, genres, and audience engagement.

The goal is to transform raw movie data into meaningful insights that support data-driven decision-making for production houses, streaming platforms, and investors.

❗ Problem Statement

The movie industry generates large volumes of data across genres, years, and performance metrics. However:

Raw data is complex and difficult to interpret
Stakeholders lack clear insights into revenue trends
Decision-making is often not data-driven

This project addresses these challenges by building a visual analytics dashboard.

🎯 Objectives
Analyze movie revenue trends over time
Understand genre-wise and year-wise performance
Identify top-performing movies and directors
Study relationships between ratings, votes, and revenue
Build an interactive and user-friendly dashboard
📂 Dataset Information
Source: CSV files
Time Period: 2006 – 2016
Data Type: Structured data
🔑 Key Features:
Movie Title
Genre
Release Year
Runtime
IMDb Rating
Votes
Metascore
Revenue (Millions)
Director
🧹 Data Cleaning & Preparation
Removed duplicate records
Handled missing and null values
Standardized column names
Converted data types
Created calculated measures using DAX
📊 Dashboard Features
🔹 1. Movie Overview & Time Trends
Year-wise movie releases
Revenue growth trends
Average rating and runtime analysis
🔹 2. Genre & Rating Analysis
Genre distribution using treemap
Revenue contribution by genre
Comparison of IMDb ratings and metascores
🔹 3. Top Performers
Highest revenue movies
Most popular movies (based on votes)
Top-performing directors
🔹 4. Correlation Analysis
Revenue vs IMDb rating (Scatter/Bubble Chart)
Runtime vs revenue trends
Genre-wise revenue over time
🔍 Key Insights
Movies with IMDb ratings around 6.8 – 7.0 tend to generate higher revenue
Some genres consistently perform better at the box office
Audience votes indicate popularity but do not guarantee high revenue
Revenue is influenced by multiple factors, not just ratings
💡 Business Applications
Helps identify profitable genres
Supports strategic decision-making for movie production
Assists in selecting high-performing directors
Enables better marketing and release planning
⚙️ Tools & Technologies
Power BI – Dashboard & Visualization
Power Query – Data Cleaning
DAX – Calculations & Measures
🚧 Limitations

No budget or profit data included
Does not cover OTT platform performance

🙌 Conclusion

This project demonstrates how Power BI can transform raw data into actionable insights. By analyzing trends in revenue, ratings, and genres, the dashboard provides valuable guidance for decision-making in the movie industry.
